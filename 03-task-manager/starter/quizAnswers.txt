2. What does CRUD stand for?
CRUD stands for Create, Read, Update, and Delete. These are the four basic operations that can be performed on data in a database or a persistent storage.

3. You are careful to store your sensitive information in a .env file.
How do you get your app to access the information in the .env file?
To access the information in a .env file, you use the dotenv package. First, you install the package using `npm install dotenv`. Then, you add `require('dotenv').config();` at the top of your main application file (e.g., app.js). This loads the environment variables from the .env file into `process.env`.
